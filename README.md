# Jahnavi Store

A Django-based E-Commerce Website developed by M. Jahnavi.

## Features

- User Registration & Login
- Product Search
- Product Categories
- Shopping Cart
- Quantity Management
- Wishlist
- Product Reviews & Ratings
- Order Placement
- Order History
- Responsive Bootstrap UI

## Technologies Used

- Python
- Django
- SQLite
- Bootstrap 5
- HTML
- CSS

## Installation

```bash
git clone <repository-url>
cd ecommerce_project

pip install django

python manage.py migrate

python manage.py runserver
```
